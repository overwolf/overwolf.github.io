    * phase
* New properties for the [GameInfo](/api/games#gameinfo-object) object:
  * monitorHandle - Returns the current monitor handle. You can now check for example, on which monitor your app is running.
  * windowHandle - Returns the current app window handle.
* New function: [overwolf.extensions.checkForExtensionUpdate()](/api/extensions#checkforextensionupdatecallback) -  allows the current app to check if there is an extension update.
