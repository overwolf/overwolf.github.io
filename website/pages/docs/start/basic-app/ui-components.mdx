In order to create a great App, more than just functionality is needed. Even the best feature in the world can be taken down by bad User Experience.  
As such, in this page, as well as throughout the entire App creation process, we will also be putting a focus on the App's visual design, responsiveness,
and general best practices.

:::tip WIP Content
This page is a work in progress. As such, more content will be added to it over time!
:::

To get you started, we have created a small figma ([?](https://www.figma.com/)) file, containing example component designs, as well
as the exact styling used to achieve them. You can check it out over [here](https://www.figma.com/file/W8BQ7CIWrPazuDOKHdTaxP/Basic-App).
